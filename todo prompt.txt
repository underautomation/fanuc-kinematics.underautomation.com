Dans RobotController.cs, modifie CalculateForwardKinematics pour retourner aussi la configuration : KinematicsUtils.ForwardKinematics retourne un type CartesianPosition qui contient les propriétés double X,Y,Z,W,P,R et la ,propriété Configuration Configuration de type (retourne ses propriétés et aussi son ToString()) : 

``` csharp
public class Configuration
{
  public WristFlip WristFlip { get; set; }
  public ArmUpDown ArmUpDown { get; set; }
  public ArmLeftRight ArmLeftRight { get; set; }
  public ArmFrontBack ArmFrontBack { get; set; }
  public int TurnAxis4 { get; set; }
  public int TurnAxis5 { get; set; }
  public int TurnAxis6 { get; set; }
}

public enum WristFlip
{
  Unknown,
  Flip,
  NoFlip,
}
public enum ArmUpDown
{
  Unknown,
  Up,
  Down,
}
public enum ArmLeftRight
{
  Unknown,
  Left,
  Right,
}
public enum ArmFrontBack
{
  Unknown,
  Front,
  Back,
}
```

- A chaque modification de joint, recalcule le forward kinematics et affiche chaque propriété à côté du TextFiels correspondant dans Sidebar. Affiche aussi la configuration

- Dans CalculateInverseKinematics, retourne toutes les solutions en joints et ajoute une pickup table où on peut selectionner la conf en joint voulue dans sidebar (un tableau en colonne les valeurs en joitn et en ligne chaque solution)
